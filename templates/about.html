About KeyLogger

KeyLogger is a tool that allows you to monitor the activity of your computer. It is a simple and easy to use tool that is designed to be used by anyone.

At KodCode, we have been instructed to develop a KeyLogger that is able to monitor the activity of the computer and send the data to a server and display it on a website.
